HSCREEN mode

Selects a high-resolution graphics screen mode.  Modes 1-4 also clear the high-resolution graphics screen.
  mode    - Mode number.  Mode numbers are:
            0 - Low resolution
            1 - 320 x 192, 4-color
            2 - 320 x 192, 16-color
            3 - 640 x 192, 2-color
            4 - 640 x 192, 4-color

Example
HSCREEN 4
