**SCREEN <span style="color:#00FFFF;">*type*</span>,<span style="color:#FFFF00;">*colors*</span>**

Selects low-resolution screen modes and color sets.

- <span style="color:#00FFFF;">type</span>
  - 0 - Text
  - 1 - Graphics
- <span style="color:#FFFF00;">colors</span>
  - 0 - Color set 0
  - 1 - Color set 1

```ecb2
10 SCREEN 0,1
```
